Salaam Anouk!

De websites zijn bijna klaar, alhamdulillah. Voor de laatste loodjes (live zetten + betalingen + bevestigingsmails) heb ik nog een paar dingen van je nodig. De meeste duren letterlijk 5 minuten — geen stress, je kunt het in stukjes doen.

Ik heb het in groepjes gezet:

**1. Domeinen — bij wie staan ze?**
Bij welke partij heb je `memoriesofdamascus.nl` en `morgenland.nu` geregistreerd (TransIP, Versio, Hostnet, Combell, GoDaddy)? Inloggegevens daar heb ik nodig om de sites op de juiste domeinen live te zetten.
`morgenlandkookt.nl` is nog niet geregistreerd — wil je dat ik dat namens jou doe (€10/jaar) of doe je het zelf?

**2. Mollie**
Voor de online betalingen heb ik nodig:
- De **Live API key** (Mollie dashboard → Developers → API keys → kopieer de live_-key)
- De **Test API key** (test_-key, op dezelfde plek)

Eén vraag erbij: in Mollie kun je per merk een apart "profiel" hebben — dat bepaalt welke naam je klant op zijn bankafschrift ziet en welk logo in het betaalscherm staat. Wat is bij jou ingesteld?
- Optie A — **1 profiel voor alles**: alle klanten (catering, koffie, workshops) zien dezelfde naam op hun afschrift (bv. "Memories of Damascus")
- Optie B — **3 aparte profielen**: catering-klant ziet "Memories of Damascus", workshop-klant ziet "Morgenland Kookt", etc. — schoner naar de klant maar wel meer setup

Als je niet zeker weet wat het is, log even in op Mollie → links **Mijn profielen** of **Profiles** → daar zie je hoeveel je er hebt. Stuur me een screenshot dan los ik het op.

**3. Bedrijfsgegevens (voor onderaan de site + privacy + voorwaarden)**
- Officiële bedrijfsnaam zoals bij KvK
- KvK-nummer
- BTW-nummer
- Vestigingsadres
- Algemeen mailadres en telefoonnummer

**4. Catalogus + voorwaarden**
- Heb je een PDF van de catalogus (gerechten + prijzen)? Stuur 'm en ik zet de download-knop live.
- Heb je al algemene voorwaarden? Zo nee, kan ik een standaard concept opstellen voor jouw soort werk — je laat 'm dan eventueel checken door een jurist.

**5. Email voor versturen**
Welk mailadres mag ik instellen als afzender voor bevestigingsmails naar je klanten? (bv. `info@memoriesofdamascus.nl`)
En waar wil je dat **jij** zelf de notificaties ontvangt als er een aanvraag binnenkomt?
